# 🏯 Shrine Database

> Frontend app for Japanese shrine data. Local JSON dataset.

## 📦 JSON Data Access

Shrine data is stored in `src/data/shrines.json`.

**Download the JSON:**

```bash
curl -o shrines.json https://raw.githubusercontent.com/webDeveloper42/main/src/data/shrines.json
```

## 🤝 Help Needed: Add Japanese Shrines

I need contributors to expand the database! If you know Japanese shrines, please help add more entries.

**How to contribute:**

1. Fork the repo
2. Edit `src/data/shrines.json` to add shrines (format: `{id, name, location, address}`)
3. Submit a PR

Current: 10 shrines. Goal: 75,000+!
